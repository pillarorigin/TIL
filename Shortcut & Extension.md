### VScode
`ctrl` + `shift` + k : 자동정렬

shift + alt + o : 정렬

`ctrl` + `shift` + v : MD file preview

`ctrl` + `k` + `R` : 단축키 레퍼런스 브라우저에서 열기

`ctrl` + `k` + `S` : 단축키 레퍼런스 vscode 안에서 열기

`ctrl` + `M` : toggel tab binding (탭으로 간격 넓히거나 탭으로 블록 잡거나 등)

[reference] : https://code.visualstudio.com/docs/languages/markdown
